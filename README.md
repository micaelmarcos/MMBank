# MMBank - Sistema Bancário em Java 🏦

O **MMBank** é um sistema de gerenciamento de contas bancárias desenvolvido como projeto de estudo. O objetivo principal é aplicar conceitos de Programação Orientada a Objetos (POO) em um cenário de transações financeiras.

## 🚀 Funcionalidades
* **Criação de Contas:** Registro de usuários com saldo inicial.
* **Depósitos e Saques:** Atualização de saldo.
* **Transferências:** Movimentação de valores entre diferentes contas cadastradas.
* **Extrato/Saldo:** Consulta rápida da situação financeira de cada conta.
* * **Saque:** Realizar saques de valores que estão na conta.
 
* ## 🛠️ Tecnologias
* **Linguagem:** Java
* **Biblioteca:** Java Swing
* **IDE:** NetBeans
* **Controle de Versão:** Git & GitHub

* ## 📂 Como testar o projeto
1. Clone este repositório.
2. Abra o projeto no **NetBeans**.
3. Certifique-se de que o JDK 17 ou superior está configurado.
4. Execute a classe principal (que contém o método `main`).
   * *Nota: Como o sistema não utiliza banco de dados, os dados são resetados sempre que o programa é encerrado.*
